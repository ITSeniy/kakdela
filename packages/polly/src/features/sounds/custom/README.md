# Пак «свой» — звуки из файлов

Кладите аудиофайлы прямо в эту папку. Имя файла = имя события,
расширение — любое из: `mp3`, `ogg`, `wav`, `m4a`, `webm`, `flac`, `aac`.

| Файл | Событие |
|---|---|
| `mute-on.*` | микрофон выключен |
| `mute-off.*` | микрофон включён |
| `deafen-on.*` | звук выключен |
| `deafen-off.*` | звук включён |
| `voice-join.*` | вы зашли в голосовой |
| `voice-leave.*` | вы вышли из голосового |
| `user-join.*` | кто-то зашёл |
| `user-leave.*` | кто-то вышел |
| `moved.*` | вас переместили |
| `notification.*` | уведомление |
| `stream-start.*` | стрим начался |
| `stream-end.*` | стрим закончился |
| `viewer-join.*` | зритель зашёл на стрим |
| `viewer-leave.*` | зритель ушёл со стрима |

Правила:

- Файлы подхватываются Vite на этапе сборки (`import.meta.glob`) —
  после добавления **перезапустите dev-сервер** или пересоберите клиент.
- События, для которых файла нет, играют синтезом из пака «какао» —
  можно заполнять папку постепенно.
- Звуки короткие (0.2–1.5 с) звучат лучше всего; громкость регулируется
  ползунком в настройках → звуки.
