# T-061: Replies (ответы на сообщения)

**Phase:** 4 — Уют
**Model:** Sonnet 4.6 · `medium`
**Estimate:** ~1.5-2 часа

## Контекст
Поле `replyToId` уже есть в схеме `messages` (с T-016). Здесь — связывание в UI: composer показывает «отвечаем на X», message рендерит цитату вверху.

## Файлы в скоупе
- `packages/polly/src/features/chat/Composer.tsx` (доработка — reply state)
- `packages/polly/src/features/chat/Message.tsx` (доработка — render reply quote, action кнопка reply)
- `packages/polly/src/features/chat/store.ts` (новый — UI state чата: replyTo)
- `packages/polly/src/features/chat/api.ts` (sendMessage уже умеет replyToId)

## Что сделать
1. **Store**:
   - `replyTo: Message | null` — установлен через `setReply(message)`, очищается после успешного send.
2. **Composer**:
   - Над input area — баннер `Отвечаете на: <author>: <preview 60 chars>` + крестик.
   - При send — передать `replyToId` в API.
3. **Message rendering**:
   - Если `message.replyToId` — над основным content рендерится мини-цитата:
     - Слева — серая вертикальная линия.
     - Сверху — мини-аватар + имя автора оригинала.
     - Под — обрезанный preview (1 строка) контента оригинала.
     - Клик по цитате → scroll к оригинальному сообщению в списке + briefly highlight.
4. **Resolve replied messages**:
   - В API ответе для message с replyToId должен быть `replyTo: { id, author, content (first line) }` (joined в SQL).
   - Если оригинал удалён — `replyTo: { deleted: true }` и UI показывает «оригинал удалён».
5. **Кнопка reply на hover сообщения** — иконка стрелки → `setReply(message)`.
6. **Scroll-to-message**:
   - Каждый Message имеет `data-message-id={message.id}`.
   - Клик по reply quote: `document.querySelector('[data-message-id="..."]')?.scrollIntoView({ block: 'center', behavior: 'smooth' })` + `el.classList.add('flash')` на 1 сек.
   - Если оригинал вне загруженной страницы — fetch предыдущих страниц до тех пор, пока не найдётся (с лимитом 5 страниц).

## DoD
- [ ] Reply кнопка → баннер появляется в composer → send → новое сообщение с цитатой
- [ ] Клик по цитате → scroll к оригиналу + flash highlight
- [ ] Reply на удалённое сообщение — показывает заглушку «оригинал удалён»
- [ ] Cancel reply (крестик) — баннер исчезает, send отправляет обычное
- [ ] `pnpm typecheck` чистый

## Out of scope
- Threads (отдельная ветка) — T-080.
- Quote-style ответы через `> ` markdown — это другое, поддерживается markdown'ом из T-021.

## Подсказки
- `flash` класс через CSS keyframe анимацию: `0% { background: var(--kd-warm-soft); } 100% { background: transparent; }` с duration 1s.
- При scrollIntoView с `behavior: smooth` НЕ запускай поиск через MutationObserver — message может ещё рендериться. Просто wait 100ms и потом scroll.
