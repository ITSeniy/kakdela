## Что сделать

1. **JS-зависимости**:
   ```
   react, react-dom, zustand, @tauri-apps/api@^2, @tauri-apps/plugin-os@^2
   ```
   Dev: `vite`, `@vitejs/plugin-react`, `@tauri-apps/cli@^2`, `typescript`, `tailwindcss`, `postcss`, `autoprefixer`, `@types/react`, `@types/react-dom`.

2. **Tauri конфиг** (`tauri.conf.json`):
   - `productName: "как дела"`, `identifier: "com.kakdela.polly"`
   - `windows[0].title: "как дела"`, размер `1280×800`, минимум `960×600`
   - Иконки — пока заглушки 32×32 / 128×128 / `.ico` (можно сгенерировать через `pnpm tauri icon путь-к-png`)
   - Bundler targets: `["msi", "deb", "appimage"]`

3. **Rust main**:
   ```rust
   fn main() {
       tauri::Builder::default()
           .plugin(tauri_plugin_os::init())
           .run(tauri::generate_context!())
           .expect("error while running tauri application");
   }
   ```
   В `Cargo.toml`: `tauri = "2"`, `tauri-plugin-os = "2"`, `serde`, `serde_json`.

4. **Tailwind config** — пробросить CSS-переменные:
   ```ts
   colors: {
     kd: {
       bg: 'var(--kd-bg)',
       'bg-deep': 'var(--kd-bg-deep)',
       panel: 'var(--kd-panel)',
       'panel-alt': 'var(--kd-panel-alt)',
       'panel-hi': 'var(--kd-panel-hi)',
       text: 'var(--kd-text)',
       'text-soft': 'var(--kd-text-soft)',
       'text-mute': 'var(--kd-text-mute)',
       accent: 'var(--kd-accent)',
       'accent-deep': 'var(--kd-accent-deep)',
       warm: 'var(--kd-warm)',
       danger: 'var(--kd-danger)',
       border: 'var(--kd-border)',
       online: 'var(--kd-online)',
       idle: 'var(--kd-idle)',
       dnd: 'var(--kd-dnd)',
     }
   },
   fontFamily: {
     sans: ['Inter', 'system-ui', 'sans-serif'],
     mono: ['"JetBrains Mono"', 'ui-monospace', 'monospace'],
   },
   borderRadius: { kd: '6px' },
   ```

5. **`theme.ts` (Zustand)**:
   - State: `theme: 'light' | 'dark' | 'system'`
   - Действие: `setTheme(t)`, при изменении выставляет `document.documentElement.dataset.theme`
   - Подписка на системную тему через `matchMedia('(prefers-color-scheme: dark)')`
   - Persist в localStorage по ключу `kd:theme`

6. **`App.tsx`** — пока минимум:
   - Шапка с лого «кд» (бежевый квадрат с инициалом)
   - Заголовок «как дела?»
   - Подзаголовок-моно «self-hosted · v0.0.1 · фаза 0»
   - Кнопка-toggle темы (компонент `ThemeToggle`)
   - Кнопка «проверить связь со speedy» — делает `fetch('http://localhost:3001/healthz')` и показывает результат в моно-шрифте
   - Всё стилизовано через Tailwind + токены, фон `bg-kd-bg`, паддинги как в дизайне

7. **`vite.config.ts`** — пути, base, `clearScreen: false` (Tauri рекомендует), `server: { port: 1420, strictPort: true }`.

8. **Скрипты** в package.json:
   ```
   "dev":      "vite",
   "build":    "tsc -b && vite build",
   "tauri":    "tauri",
   "tauri:dev":"tauri dev",
   "tauri:build":"tauri build",
   "typecheck":"tsc --noEmit"
   ```

## Acceptance criteria

- [ ] `pnpm dev:web` → открывается `http://localhost:1420` с приветственным экраном
- [ ] `pnpm dev:polly` → открывается окно Tauri с тем же экраном
- [ ] Переключение тема (клик ↑ или ⌘+T) меняет цвета мгновенно
- [ ] Кнопка «проверить связь» при включённом speedy показывает зелёный `ok`, при выключенном — красный `unreachable`
- [ ] `pnpm typecheck` чистый
- [ ] Окно Tauri не моргает белым при старте (выставить `backgroundColor` в `tauri.conf.json` под тему по умолчанию)

## Подсказки

- На Windows может не быть установлен WebView2 Runtime — на чистой Win10 нужно поставить. Win11 — из коробки.
- На Linux под Wayland Tauri иногда мерцает — добавь `WEBKIT_DISABLE_DMABUF_RENDERER=1` в env.
- Иконки: сгенерируй один `app-icon.png` 1024×1024, потом `pnpm tauri icon path/to/app-icon.png` создаст всё, что нужно.
- В дизайне `KD_FONT = "Inter"` и `KD_MONO = "JetBrains Mono"` — добавь их через `@fontsource/inter` и `@fontsource/jetbrains-mono`, импортируй в `main.tsx`. Так не зависим от системных шрифтов.
- Тема персистится в localStorage. Тауриспецифики тут нет, обычный браузерный API в WebView2.
