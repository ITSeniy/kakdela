# T-034: UI голосового канала (без screen share)

**Phase:** 2 — Голос
**Model:** Sonnet 4.6 · `medium`
**Estimate:** ~3 часа

## Контекст
Дизайн в `designs/final-voice.jsx`. На этом этапе НЕ делаем screen share — только tile'ы с avatar'ами и индикацией speaking. Screen share — фаза 3 (T-050+).

## Файлы в скоупе
- `packages/polly/src/features/voice/VoiceScreen.tsx` (новый)
- `packages/polly/src/features/voice/ParticipantTile.tsx` (новый)
- `packages/polly/src/features/voice/VoiceControls.tsx` (новый — bottom bar)
- `packages/polly/src/app/Router.tsx` (роут для voice-каналов)

## Что сделать
1. **Роут**: `/servers/:serverId/channels/:channelId` с проверкой `channel.kind`. Если 'voice' — `<VoiceScreen>`, если 'text' — `<ChatScreen>`.
2. **`VoiceScreen.tsx`**:
   - Заголовок канала сверху как в text-канале (имя, topic).
   - Если user НЕ подключён: большой центральный CTA «Подключиться» — клик → `useVoiceRoom().join()`.
   - Если подключён: grid с `<ParticipantTile>` для каждого участника (включая себя).
   - Grid layout: 1 участник — full screen; 2 — 1×2; 3-4 — 2×2; 5-9 — 3×3; 10+ — 4×N с auto-fit.
   - Внизу — `<VoiceControls>`.
3. **`ParticipantTile.tsx`**:
   - Avatar большой по центру (или camera/screen tile в фазе 3).
   - Имя внизу.
   - Кольцо вокруг tile'а зелёное (`--kd-online`), если activeSpeaker.
   - Иконка mute crossed на иконке микрофона в углу.
   - Аудио элементы (`<audio autoplay>`) добавлять/удалять при subscribe/unsubscribe — это делается через LiveKit `attach()` / `detach()` API.
4. **`VoiceControls.tsx`** (sticky bottom):
   - Кнопка mute (микрофон / микрофон-crossed).
   - Кнопка deafen (наушники / наушники-crossed).
   - Кнопка «демо» — заглушка кликабельная, в T-050 включится логика.
   - Кнопка «выйти» — большая красная, leaveChannel.
   - Слева — индикатор RTT (badge с моно-цифрами «42 мс»).
5. **Сайдбар голосового канала**: в `ChannelList.tsx` (из T-018) под voice-каналом, в который кто-то подключён, показываем мини-список аватарок участников (из webhook через WS). Клик по аватарке → переход в этот канал.

## Подсказки
- LiveKit `track.attach()` возвращает `HTMLMediaElement`. Не создавай `<audio>` руками — используй вернутый элемент.
- Grid layout удобнее всего сделать через CSS grid с `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))` для 5+ участников. Для 1-4 — явная заготовка (1, 2, 2x2).
- `activeSpeakers` от LiveKit прилетает с дебаунсом ~500ms — слишком резкое моргание неприятно глазу. Это OK, ничего не делать.
