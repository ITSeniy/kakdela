# T-081: Custom emoji

**Phase:** 5
**Model:** Sonnet 4.6 · `medium`
**Estimate:** ~2-3 часа

## Контекст
Custom emoji — это server-specific images, доступные через `:name:` syntax. Хранятся в отдельном MinIO bucket для cacheability и публичной отдачи.

## Файлы в скоупе
- `packages/speedy/src/db/schema.ts` (таблица `emoji`)
- `packages/speedy/src/routes/emoji.ts` (новый)
- `packages/polly/src/features/chat/markdown.ts` (доработка — emoji replacement)
- `packages/polly/src/features/chat/EmojiPicker.tsx` (доработка — custom tab)
- `packages/polly/src/features/settings/EmojiManagement.tsx` (новый — для admin'ов)

## Что сделать
1. **Таблица `emoji`**:
   - id, serverId fk, name text (unique per server), imageUrl, animated boolean, uploadedBy fk, createdAt.
2. **Эндпоинты**:
   - `GET /api/servers/:serverId/emoji` → список (для members).
   - `POST /api/servers/:serverId/emoji` (admin) → upload + create. Тело: `{ name, file }`. Файл: max 256 KB, PNG/GIF, max 128×128.
   - `DELETE /api/emoji/:id` (admin).
   - Storage: отдельный bucket `kakdela-emoji` с public-read для cacheability.
3. **Markdown**:
   - В `mentions.ts` / `markdown.ts` после mentions добавить шаг — `:emoji_name:` → `<img class="kd-emoji" src="..." alt=":emoji_name:">`.
   - lookup в локальном кэше emoji (TanStack Query, per-server).
4. **EmojiPicker custom tab**:
   - Tab «Сервер» в picker'е → grid из server's custom emoji.
   - Tooltip на hover показывает `:name:`.
   - Клик вставляет `:name:` в composer.
5. **EmojiManagement** (settings → server):
   - Admin grid с upload form (drag-n-drop), edit name, delete.
   - Limit: 50 emoji per server (default, configurable через env).

## DoD
- [ ] Admin загружает PNG → emoji появляется в picker'е сервера → можно использовать
- [ ] `:custom_emoji:` в сообщении рендерится как картинка
- [ ] Member не может загрузить (403)
- [ ] Удаление работает, существующие сообщения с использованием — продолжают показывать (cached) но в picker'е нет
- [ ] `pnpm typecheck` чистый

## Out of scope
- Server emoji slots / нитро-tier — у нас всё бесплатно.
- Animated emoji APNG — поддерживаем GIF.
- Sticker pack — нет.

## Подсказки
- Public bucket — настрой через MinIO mc CLI на старте: `mc anonymous set download local/kakdela-emoji`. Альтернатива: presigned GET через speedy (но кэшировать сложнее).
- При render emoji в сообщении — НЕ делай N запросов на /emoji/:id. Загружай весь список per-server один раз и держи в memory.
