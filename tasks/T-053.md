# T-053: Снапшот кадра демонстрации → в чат

**Phase:** 3 — Демонстрация экрана
**Model:** Sonnet 4.6 · `medium`
**Estimate:** ~1.5 часа

## Контекст
Уютная фича: «о, прикольный момент в твоей демке!» → клик → текущий кадр улетает в text-канал того же сервера как картинка. Своеобразный mini-clip.

## Файлы в скоупе
- `packages/polly/src/features/voice/ScreenTile.tsx` (кнопка snapshot)
- `packages/polly/src/features/voice/snapshot.ts` (новый — canvas capture)
- `packages/polly/src/features/files/upload.ts` (новый — заглушка под T-063, или сразу полноценный)
- `packages/speedy/src/routes/files.ts` (минимальный endpoint для upload)

## Что сделать
1. **`snapshot.ts`**:
   ```
   async function snapshotTrack(track: VideoTrack): Promise<Blob> {
     const el = track.attach()   // временный <video>
     const canvas = document.createElement('canvas')
     canvas.width = el.videoWidth
     canvas.height = el.videoHeight
     canvas.getContext('2d').drawImage(el, 0, 0)
     track.detach(el)
     return new Promise(res => canvas.toBlob(res, 'image/jpeg', 0.85))
   }
   ```
2. **Кнопка в ScreenTile** (onhover):
   - Иконка камеры + tooltip «снимок в чат»
   - Клик → `snapshotTrack(track)` → upload в MinIO (через speedy presigned POST) → отправить сообщение в default text-канал сервера с attachment'ом
3. **Минимальный upload-флоу** (это пересекается с T-063, можно сделать здесь упрощённую версию):
   - Клиент: POST `/api/files/presign` `{ contentType: 'image/jpeg', size: blob.size }` → ответ `{ uploadUrl, fileId, publicUrl }`.
   - Клиент: PUT blob на uploadUrl напрямую в MinIO (с правильными headers).
   - Клиент: POST `/api/channels/:id/messages` с `content: ''` и `attachments: [fileId]`.
4. **Speedy `routes/files.ts`**:
   - `POST /api/files/presign` — генерирует MinIO presigned URL (SDK `@aws-sdk/client-s3` + `@aws-sdk/s3-request-presigner`), key = `<userId>/<uuid>.<ext>`, TTL = 5 минут.
   - Возвращает publicUrl как `<S3_ENDPOINT>/<S3_BUCKET>/<key>`.
5. **Куда отправить**: текущий канал, в котором был открыт voice (если open) — или в первый text-канал сервера. UI должен показать toast «снимок отправлен в #general».

- Видео-клипы 10s — нет.
- Поддержка attachments в полной мере — T-063.

## Подсказки
- Если video track в момент snapshot ещё не имеет dimensions (ready) — drawImage даст пустой canvas. Проверь `videoWidth > 0` перед.
- `image/webp` качественнее при том же размере, но не все CDN/messengers умеют отдавать. JPEG — безопаснее.
- MinIO presign требует, чтобы клиент послал **точно** те же headers что были подписаны. Минимум `Content-Type`. Проблема — нельзя послать дополнительные headers, поэтому presign без CORS.
